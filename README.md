# Online Course Enrollment System

## Project Description
This is a web-based online course enrollment system developed using PHP and MySQL.

## Features
- Student registration and login
- Course enrollment
- Admin panel
- Course management
- Database connectivity

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## How to Run
1. Install WAMP/XAMPP
2. Place project inside www or htdocs folder
3. Import database in phpMyAdmin
4. Start Apache and MySQL
5. Open project in browser
